🛡️ RedSpecter – Linux Privilege Escalation Framework
📖 Overview

RedSpecter is a Linux Privilege Escalation framework designed to automate the
enumeration of potential misconfigurations, SUID binaries, writable files,
and other privilege escalation vectors.

It is intended for:

Penetration testers
Red teamers
Cybersecurity learners

⚠️ Disclaimer

This tool is developed strictly for educational and authorized penetration testing purposes only.
Do not use it on systems without proper permission.

🚀 Features

Automated system enumeration
SUID/SGID binary detection
Writable file and directory checks
Cron job analysis
Environment variable inspection
Kernel version check

🖥️ Requirements

Linux system
Bash support
Basic user privileges

📥 Installation :-
git clone https://github.com/shaikhammu16092004-web/RedSpecter-Linux-PrivEsc-Framework.git
cd RedSpecter-Linux-PrivEsc-Framework
chmod +x privesc.sh

▶️ RUN :-
bash privesc.sh
